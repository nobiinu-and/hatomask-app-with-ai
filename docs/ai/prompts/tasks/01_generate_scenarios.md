# Gherkinシナリオ生成

## 依頼内容
以下の機能仕様書(Spec)を基に、Cucumberで使用するGherkin形式のシナリオを作成してください。

## 入力情報
- **Specファイル**: `[Specファイルのパス]`
- **対象機能**: `[機能名]`

## 制約事項
1. **言語**: 日本語で記述すること。
2. **フォーマット**:
   ```gherkin
   Feature: [機能名]
     Scenario: [シナリオ名]
       Given [前提条件]
       When [操作]
       Then [期待結果]
   ```
3. **網羅性**:
   - 正常系 (Happy Path)
   - 異常系 (Error Cases)
   - 境界値 (Edge Cases)
4. **粒度**: 実装の詳細ではなく、ユーザーの振る舞いに焦点を当てること。

## 出力形式
作成したシナリオをMarkdownのコードブロックで出力してください。
